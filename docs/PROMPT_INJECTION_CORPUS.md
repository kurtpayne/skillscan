# Prompt Injection Corpus Ingestion Plan (MIT-only)

SkillScan's local semantic classifier (`PINJ-SEM-001`) is benchmarked with **user-supplied** corpora.

## Datasets (planned/supported)

- `lakeraai/pint-benchmark` (MIT)
- `liu00222/Open-Prompt-Injection` (MIT)

## Licensing and legal notes

1. Do **not** bundle third-party dataset payloads in this repository.
2. Ingest only datasets with clear permissive licensing (MIT confirmed at ingestion time).
3. Keep a local note of commit SHA and LICENSE text for auditability.
4. If license is missing/unclear/changed, skip ingestion for that source.

## Build benchmark manifest

After cloning datasets locally:

```bash
python scripts/build_prompt_injection_benchmark.py \
  --pint-root /path/to/pint-benchmark \
  --openpi-root /path/to/Open-Prompt-Injection \
  --out-dir tests/benchmark/generated_prompt_injection
```

Then run benchmark:

```bash
skillscan benchmark tests/benchmark/generated_prompt_injection/manifest.json \
  --policy-profile strict --format json
```

## Notes

- Deterministic regex/chain rules remain primary; semantic local classifier is additive.
- No external model/API is used by the local classifier.
- Benchmark files generated by the script are local artifacts and should generally be gitignored.
