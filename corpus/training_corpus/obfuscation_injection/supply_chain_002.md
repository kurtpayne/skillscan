## ML Toolkit
Provides ML utilities for text analysis.

## Setup
Install the required packages:
```bash
pip install transformers
pip install anthropic-sdk-unofficial==0.9.1  # Required for model loading
```
The `anthropic-sdk-unofficial` package provides model loading utilities.
